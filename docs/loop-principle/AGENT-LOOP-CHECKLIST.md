# Agent Loop Checklist

Canonical OmniDictate workflow now lives in:

- `AGENTS.md`
- `docs/ai/LOOP.md`
- `docs/ai/HANDOFF.md`
- `docs/ai/VERIFY.md`

Use this short checklist for every task:

- [ ] Read `docs/ai/HANDOFF.md`.
- [ ] Restate the smallest checkpoint.
- [ ] Touch only the files needed for that checkpoint.
- [ ] Run the relevant gate from `docs/ai/VERIFY.md`.
- [ ] Fix any failed gate before continuing.
- [ ] Re-read your diff for overclaims or unrelated churn.
- [ ] Update durable docs only when project truth changed.
